# Mini.nvim設定

このディレクトリには[mini.nvim](https://github.com/echasnovski/mini.nvim)（Neovim用のミニマルプラグインコレクション）の設定ファイルが含まれています。

## ファイル構成

- `init.lua`: mini.nvimの初期化スクリプト

## 概要

mini.nvimは、20以上の独立したNeovimプラグインを提供するライブラリです。
このディレクトリには、mini.nvimのモジュールを初期化するための設定が含まれています。
