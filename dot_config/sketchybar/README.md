# SketchyBar設定

このディレクトリには[SketchyBar](https://github.com/FelixKratz/SketchyBar)（macOS向けカスタマイズ可能なステータスバー）の設定ファイルが含まれています。

## ファイル構成

- `sketchybarrc`: SketchyBarのメイン設定ファイル
- `plugins/`: SketchyBarのプラグインスクリプト

## 概要

SketchyBarは、macOSのメニューバーを置き換えるカスタマイズ可能なステータスバーです。
システム情報やカスタムスクリプトの出力を表示できます。
