# Zoxide設定

このディレクトリには[Zoxide](https://github.com/ajeetdsouza/zoxide)（スマートなcdコマンド代替）の設定ファイルが含まれています。

## ファイル構成

- `zoxide.nu`: Nushell用のZoxide統合スクリプト

## 概要

Zoxideは、頻繁にアクセスするディレクトリを学習し、部分一致で素早く移動できるcdコマンドの代替ツールです。
このディレクトリには、Nushellとの統合設定が含まれています。

## 使用方法

Zoxideを使用すると、完全なパスを入力せずにディレクトリに移動できます。
例えば、`z doc`と入力すると、履歴から最も頻繁にアクセスする`documents`のようなディレクトリに移動します。
