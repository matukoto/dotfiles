# Nushell設定

このディレクトリには[Nushell](https://www.nushell.sh/)（新しいタイプのシェル）の設定ファイルが含まれています。

## ファイル構成

- `config.nu.tmpl`: Nushellのメイン設定ファイル（chezmoiテンプレート）
- `env.nu`: Nushellの環境変数設定

## 概要

Nushellは、構造化データを扱うことに特化した現代的なシェルです。
パイプラインで表形式データを扱い、クロスプラットフォームで動作します。

## 主な設定内容

### テーマ

デフォルトテーマの色設定が定義されています。

### 補完

- 外部コマンドのカスタム補完
- `nu`コマンド自体の補完設定

### カスタムコマンド

設定ファイル内でNushell用のカスタムコマンドが定義されています。
