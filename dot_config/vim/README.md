# Vim設定

このディレクトリにはVim（テキストエディタ）の追加設定ファイルが含まれています。

## ファイル構成

- `vimrc`: Vimの追加設定ファイル
- `gitmessage`: Gitコミットメッセージのテンプレート
- `cheatSheet/`: チートシート
- `dadbod-ui/`: vim-dadbod-ui（データベースUI）の設定
- `sonictemplate/`: SonicTemplateのテンプレートファイル

## 概要

このディレクトリには、Vimで使用される追加の設定やテンプレートが含まれています。
メインのVim設定は`~/.vimrc`に配置されています。
