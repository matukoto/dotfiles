# AI エージェント設定

## 基本ルール

日本語でやりとりすること。
やりとりを要約し

- 依頼されたこと
- 計画したこと
- 実行したこと
- 検証したこと
などを docs/agents/ 以下に保存すること。

### ファイル名

`{日本語の作業内容}.md`

## 設計作業ルール

設計作業を依頼された場合は、以下のルールに従ってファイルを作成すること。

### ファイル名

`{日本語の作業内容}.md`

### 保存場所

`docs/` 以下の **作業カテゴリ別ディレクトリ** に保存する。

カテゴリ例：

- `design/`（設計・要件）
- `research/`（調査・レビュー）

### フォーマット

Markdown（拡張子 `.md`）

### ファイル作成例

```text
docs/design/ユーザー認証システム設計.md
docs/research/画像最適化手法調査.md
```
