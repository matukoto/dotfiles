# Yazi設定

このディレクトリには[Yazi](https://github.com/sxyazi/yazi)（ターミナルファイルマネージャー）の設定ファイルが含まれています。

## ファイル構成

- `yazi.toml`: Yaziのメイン設定ファイル
- `keymap.toml`: キーバインド設定
- `theme.toml`: テーマ設定

## 概要

Yaziは、非同期I/Oを活用した高速なターミナルファイルマネージャーです。
Vim風のキーバインドとプレビュー機能を備えています。
