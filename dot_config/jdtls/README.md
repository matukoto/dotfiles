# JDTLS設定

このディレクトリには[Eclipse JDT Language Server](https://github.com/eclipse-jdtls/eclipse.jdt.ls)（Java向けLanguage Server）の設定ファイルが含まれています。

## ファイル構成

- `google-java-format.xml`: Google Java Formatのコードスタイル設定

## 概要

JDTLSは、JavaのためのLanguage Server Protocol実装です。
このディレクトリには、Javaコードのフォーマットに使用するGoogle Java Styleの設定が含まれています。
