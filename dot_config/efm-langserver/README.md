# efm-langserver設定

このディレクトリには[efm-langserver](https://github.com/mattn/efm-langserver)（汎用言語サーバー）の設定ファイルが含まれています。

## ファイル構成

- `config.yaml`: efm-langserverのメイン設定ファイル

## 概要

efm-langserverは、外部のリンター、フォーマッターをLanguage Server Protocolに統合するための汎用言語サーバーです。
複数の言語やツールを1つの言語サーバー経由で管理できます。
