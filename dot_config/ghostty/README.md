# Ghostty設定

このディレクトリには[Ghostty](https://ghostty.org/)（高速なターミナルエミュレータ）の設定ファイルが含まれています。

## ファイル構成

- `config`: Ghosttyのメイン設定ファイル

## 主な設定内容

### フォント

- フォントファミリー: `HackGen Console NF`
- フォントサイズ: 16

### クリップボード

- 選択時自動コピー: 有効
- クリップボード読み取り: 確認（ask）
- クリップボード書き込み: 許可（allow）
- 末尾空白の自動削除: 有効

### ウィンドウ

- ウィンドウサイズ: 幅120×高さ40
- タイトルバー表示: 有効（macOSの信号機ボタン表示）
- 最後のタブが閉じられた時にアプリを終了: 有効
- 閉じる確認: 無効
