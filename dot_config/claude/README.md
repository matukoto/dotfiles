# Claude設定

このディレクトリにはClaude Desktop（Anthropic社のAIアシスタント）の設定ファイルが含まれています。

## ファイル構成

- `settings.json`: Claude Desktopの設定ファイル
- `CLAUDE.md`: Claude Desktop使用時のガイドライン

## 概要

Claude Desktopは、Anthropic社が提供するAIアシスタントのデスクトップアプリケーションです。
このディレクトリには、その動作設定やModel Context Protocol (MCP)の設定が含まれています。

## 主な設定内容

### MCP (Model Context Protocol)

`settings.json`には、MCPサーバーの設定が含まれています。
MCPは、Claudeが外部のツールやデータソースと連携するためのプロトコルです。

### カスタムガイドライン

`CLAUDE.md`には、Claude使用時の特定のガイドラインや指示が記載されています。
