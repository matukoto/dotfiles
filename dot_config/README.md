# dot_config

`~/.config`以下に配置される設定ファイル群です。

-   **[aerospace](./aerospace/)**: macOS用のタイリングウィンドウマネージャの設定。
-   **[aquaproj-aqua](./aquaproj-aqua/)**: CLIツールのバージョン管理ツール`aqua`の設定。
-   **[atuin](./atuin/)**: シェル履歴を管理する`atuin`の設定。
-   **[claude](./claude/)**: AIアシスタント`claude`の設定。
-   **[efm-langserver](./efm-langserver/)**: LSPを通じてLinter/Formatterを統一的に扱う`efm-langserver`の設定。
-   **[fish](./fish/)**: `fish`シェルの設定、関数、補完など。
-   **[ghostty](./ghostty/)**: ターミナルエミュレータ`ghostty`の設定。
-   **[git](./git/)**: `git`のグローバルなignoreファイル。
-   **[gitui](./gitui/)**: `git`のTUIクライアント`gitui`のキーバインド設定。
-   **[jdtls](./jdtls/)**: JavaのLanguage Server`jdtls`の設定。
-   **[mini](./mini/)**: `mini.nvim` (Neovimディストリビューション)の設定。
-   **[mise](./mise/)**: プロジェクトごとの開発環境を管理する`mise`の設定。
-   **[nushell](./nushell/)**: `nushell`の設定と環境変数。
-   **[nvim](./nvim/)**: Neovimの設定。`init.lua`を起点に、`lua/`以下にプラグインやLSPの設定を記述しています。
-   **[private_jj](./private_jj/)**: `jj` (Jujutsu) のプライベートな設定。
-   **[sketchybar](./sketchybar/)**: macOS用のステータスバー`sketchybar`の設定。
-   **[sqls](./sqls/)**: SQLのLanguage Server`sqls`の設定。
-   **[starship.toml](./starship.toml)**: クロスシェル対応のプロンプト`starship`の設定。
-   **[stylua](./stylua/)**: Luaのフォーマッタ`stylua`の設定。
-   **[typos](./typos/)**: ソースコードのタイポをチェックする`typos`の設定。
-   **[vim](./vim/)**: Vimの設定。
-   **[wezterm](./wezterm/)**: ターミナルエミュレータ`wezterm`の設定。
-   **[yamlfmt](./yamlfmt/)**: YAMLのフォーマッタ`yamlfmt`の設定。
-   **[yazi](./yazi/)**: ターミナルベースのファイルマネージャ`yazi`の設定。
-   **[zoxide](./zoxide/)**: ディレクトリ移動を高速化する`zoxide`の設定。
