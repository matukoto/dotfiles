# WezTerm設定

このディレクトリには[WezTerm](https://wezfurlong.org/wezterm/)（高機能なターミナルエミュレータ）の設定ファイルが含まれています。

## ファイル構成

- `wezterm.lua`: WeziTermのメイン設定ファイル
- `keybinds.lua`: キーバインド設定

## 主なキーバインド

### ワークスペース操作

- `LEADER + w`: ワークスペース一覧を表示して切り替え
- `LEADER + $`: 現在のワークスペースの名前を変更
- `LEADER + SHIFT + W`: 新しいワークスペースを作成

### その他

設定ファイルには、ウィンドウ操作やタブ管理など、さまざまなキーバインドが定義されています。
詳細は`keybinds.lua`を参照してください。

## 機能

- **ステータス表示**: アクティブなキーテーブル名を右側のステータスエリアに表示
- **ワークスペース管理**: 複数のワークスペースを管理し、簡単に切り替え可能

## 概要

WeziTermは、GPUアクセラレーションを活用した高速なターミナルエミュレータです。
クロスプラットフォームで動作し、豊富なカスタマイズ機能を備えています。
