# Jujutsu設定

このディレクトリには[Jujutsu (jj)](https://martinvonz.github.io/jj/)（Git互換のバージョン管理システム）の設定ファイルが含まれています。

## ファイル構成

- `config.toml`: Jujutsuのメイン設定ファイル

## 概要

Jujutsu (jj) は、Gitと互換性のある新しいバージョン管理システムです。
より直感的で安全な操作モデルを提供し、Gitリポジトリと相互運用できます。

## 注意

このディレクトリは`private_jj`という名前であり、chezmoiによって管理される際、プライベートな設定として扱われる可能性があります。
